ionic material demo!

get ready
-------------

you will need:
>- nodejs https://nodejs.org/en/
>- ionic http://ionicframework.com/

 **clone the repo **

open nodejs and from  the root folder run these commands .
>- npm install
>- bower install
>- ionic serve --lab
